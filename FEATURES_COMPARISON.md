# Firefly 功能对比和改进计划

## 已实现的基础功能 ✅

- ✅ 文章管理（CRUD）
- ✅ 分类和标签系统
- ✅ 评论系统
- ✅ 音乐播放器（基础功能）
- ✅ 暗色模式
- ✅ 响应式设计
- ✅ 用户认证

## Firefly 特色功能（待实现）📋

### 1. 主题定制功能
- [ ] **主题色自定义** - 360°色相调节，支持亮色/暗色/跟随系统
- [ ] **壁纸模式切换** - 横幅壁纸、全屏壁纸、纯色背景，前台一键切换
- [ ] **布局切换** - 列表/网格布局，前台自由切换
- [ ] **字体管理** - 支持自定义字体，丰富的字体选择器

### 2. 视觉效果
- [ ] **樱花特效** - 可配置数量的樱花飘落动画
- [ ] **看板娘** - 支持 Spine 和 Live2D 两种动画引擎
- [ ] **文章随机封面图** - 支持通过 API 获取随机封面图

### 3. 内容增强
- [ ] **增强代码块** - 基于 Expressive Code，支持代码折叠、行号、语言标识
- [ ] **数学公式支持** - KaTeX 渲染引擎，支持行内和块级公式
- [ ] **图片灯箱** - Fancybox 图片预览功能
- [ ] **文章目录** - 自动生成，支持桌面端和移动端

### 4. 音乐播放器增强
- [ ] **基于 APlayer** - 更强大的音乐播放器
- [ ] **歌词显示** - 支持 LRC 歌词文件
- [ ] **播放模式** - 单曲循环、列表循环、随机播放
- [ ] **Meting API 支持** - 在线音乐源

### 5. 其他功能
- [ ] **RSS 订阅** - 自动生成 RSS Feed
- [ ] **站点地图** - 自动生成 XML Sitemap
- [ ] **友情链接** - 精美的友链展示卡片
- [ ] **访问量统计** - 更详细的访问统计
- [ ] **评论系统集成** - 支持 Twikoo、Waline、Giscus、Disqus
- [ ] **追番页** - 基于 Bangumi API 的追番记录展示

## 改进建议

当前系统已经实现了核心功能，但可以进一步参考 Firefly 的设计理念，添加更多个性化和用户体验优化的功能。
