# 博客系统前端

基于 Vue3 + Vite + Element Plus 的博客系统前端。

## 环境要求

- Node.js 16+
- npm 或 pnpm

## 安装和运行

1. 安装依赖：
```bash
npm install
# 或
pnpm install
```

2. 启动开发服务器：
```bash
npm run dev
# 或
pnpm dev
```

3. 构建生产版本：
```bash
npm run build
# 或
pnpm build
```

## 功能特性

- 响应式设计
- 暗色模式支持
- 文章列表和详情
- 分类和标签筛选
- 评论功能
- 音乐播放器
- 管理后台

## 技术栈

- Vue 3
- Vite
- Vue Router
- Pinia
- Element Plus
- Tailwind CSS
- Marked (Markdown渲染)
